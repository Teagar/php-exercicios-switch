# Exercícios PHP

## Objetivo
Praticar os seguintes conceitos de PHP usando apenas o terminal:
- `readline()`      entrada de dados pelo terminal
- `switch`          estrutura de decisão (obrigatório em todos)
- `if/else`         validações simples (sem elseif!)
- Concatenação de strings (`.` e `.=`)
- Operações aritméticas (`+`, `-`, `*`, `/`, `%`)

## Como executar
1. Clone este repositório:
   ```bash
   git clone https://github.com/teagar/php-exercicios-switch.git
   cd php-exercicios-switch
    ```

## Regras

- Usar `switch` em TODOS os exercícios
- Usar if/else quando necessário (validações)
- NÃO usar `elseif`
- Usar readline() para entrada de dados
- Usar concatenação (.) para montar as saídas
- Usar operações aritméticas nos cálculos

## Conceitos-chave

- `readline("mensagem")`      lê input do usuário no terminal
- `(int)` ou `(float)`       converte string para número
- `.`                         operador de concatenação
- `switch($variavel) { case 'valor': ... break; }`  decisão múltipla
- `if/else` decisão binária (sem `elseif`)
